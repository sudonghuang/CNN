```python
# tests/unit/test_face_detector.py
def test_detect_single_face(face_detector, sample_face_image):
    """正常人脸图片应检测到1个人脸"""
    bboxes = face_detector.detect(sample_face_image)
    assert len(bboxes) == 1
    assert bboxes[0].confidence >= 0.9
    assert bboxes[0].x2 > bboxes[0].x1
    assert bboxes[0].y2 > bboxes[0].y1

def test_detect_no_face(face_detector, blank_image):
    """纯色图片应返回空列表"""
    bboxes = face_detector.detect(blank_image)
    assert bboxes == []

def test_detect_multiple_faces(face_detector, group_photo):
    """多人合影应检测到多个人脸"""
    bboxes = face_detector.detect(group_photo)
    assert len(bboxes) >= 2
```

## 4. AI 模型评估方案

### 4.1 评估数据集划分

```
自建校园数据集（假设50人，每人10张）：
  ├─ 训练集：每人7张（70%）
  ├─ 验证集：每人2张（20%）
  └─ 测试集：每人1张（10%，不参与任何训练决策）

评估时额外测试：
  ├─ 不同光照条件的图片
  ├─ 不同角度（侧脸）图片
  └─ 戴眼镜/口罩的图片（鲁棒性测试）
```

### 4.2 评估指标

| 指标 | 公式 | 目标值 |
|------|------|--------|
| Top-1 准确率 | 正确识别数 / 总测试数 | ≥ 85% |
| 精确率（Precision） | TP / (TP + FP) | ≥ 0.85 |
| 召回率（Recall） | TP / (TP + FN) | ≥ 0.85 |
| F1 Score | 2 × P × R / (P + R) | ≥ 0.85 |
| AUC-ROC | ROC 曲线下面积 | ≥ 0.90 |
| 单帧推理时间 | ms/帧（含检测+提取+匹配） | ≤ 200ms（CPU） |
| 误识率（FAR） | FP / (FP + TN) | ≤ 5% |
| 拒识率（FRR） | FN / (FN + TP) | ≤ 10% |

### 4.3 VGG16 vs ResNet50 对比实验

| 实验维度 | VGG16 | ResNet50 | 说明 |
|----------|-------|----------|------|
| 参数量 | 138M | 25M | ResNet50 更轻量 |
| 特征维度 | 512d | 512d（avgpool后） | 统一维度对比 |
| 训练时长（30 epoch） | 待测 | 待测 | GPU 下记录 |
| Top-1 准确率 | 待测 | 待测 | 在测试集上 |
| CPU 推理延迟 | 待测 | 待测 | ms/帧 |
| 内存占用（推理） | 待测 | 待测 | MB |

> **结论预期**：ResNet50 在准确率相近的情况下，推理速度更快、参数更少，适合部署为默认模型。

### 4.4 混淆矩阵分析

对测试集每个学生生成混淆矩阵，重点检查：

- **高混淆对**（误识率最高的两人）：分析其特征向量的余弦相似度
- **低召回学生**：通常因注册图片质量差（光线不足、遮挡）

### 5.2 AI 推理性能测试

```python
# tests/performance/test_inference_speed.py
import time

def test_single_frame_inference_time():
    """单帧推理时间应 <= 200ms（CPU环境）"""
    model_manager = ModelManager.get_instance()
    test_image = load_test_image()
    
    times = []
    for _ in range(100):  # 100次取均值
        start = time.perf_counter()
        results = model_manager.recognize_faces(test_image)
        elapsed = (time.perf_counter() - start) * 1000
        times.append(elapsed)
    
    avg_time = sum(times) / len(times)
    p99_time = sorted(times)[98]
    
    assert avg_time <= 200, f"平均推理时间 {avg_time:.1f}ms 超出目标"
    assert p99_time <= 500, f"P99推理时间 {p99_time:.1f}ms 超出目标"
```

## 6. 安全测试方案

| 测试项 | 测试方法 | 通过标准 |
|--------|----------|----------|
| SQL 注入防护 | 在查询参数中注入 `' OR 1=1--` | 返回 400，不执行注入 |
| JWT 伪造 | 使用错误密钥签名的 Token 访问接口 | 返回 401 |
| 越权访问 | 学生 Token 访问管理员接口 | 返回 403 |
| 文件上传类型绕过 | 上传重命名为 `.jpg` 的 PHP 文件 | 返回 400，内容类型校验失败 |
| 密码暴力破解防护 | 连续10次错误登录 | （P2）触发限流或账号锁定 |
| 敏感信息泄露 | 检查 500 错误响应是否包含堆栈信息 | 生产模式下不暴露 traceback |
