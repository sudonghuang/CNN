## 1. 用户认证模块详细设计

### 1.1 登录接口（POST /api/auth/login）

**处理流程：**

```
输入: { username, password }
  │
  ├─ 1. 参数校验：username/password 非空，长度限制
  ├─ 2. 查询 users 表：SELECT * FROM users WHERE username=?
  ├─ 3. 若账号不存在 → 返回 401（不区分账号/密码，防枚举）
  ├─ 4. 若 is_active=0 → 返回 403
  ├─ 5. bcrypt.checkpw(password, password_hash) 验证
  ├─ 6. 失败 → 返回 401
  └─ 7. 成功 → 生成 JWT：
           payload = { sub: user.id, role: user.role, exp: now+24h }
           token = jwt.encode(payload, SECRET_KEY, algorithm='HS256')
           返回 { token, user_info }
```

**JWT Payload 结构：**

```json
{
  "sub": 1,
  "role": "teacher",
  "real_name": "张三",
  "exp": 1713322800,
  "iat": 1713236400
}
```

**路由守卫装饰器实现要点：**

```python
# app/utils/auth.py
def require_roles(*roles):
    """角色权限装饰器，用法：@require_roles('admin', 'teacher')"""
    def decorator(f):
        @wraps(f)
        def wrapper(*args, **kwargs):
            token = request.headers.get('Authorization', '').replace('Bearer ', '')
            payload = jwt.decode(token, SECRET_KEY, algorithms=['HS256'])
            if payload['role'] not in roles:
                abort(403)
            g.current_user = payload
            return f(*args, **kwargs)
        return wrapper
    return decorator
```

### 4.2 识别去重逻辑

```python
# 内存中维护已识别集合，任务结束后清理
recognized_students: Dict[int, Set[str]] = {}  # task_id → set(student_id)

def record_if_new(task_id: int, student_id: str, confidence: float):
    if student_id in recognized_students.get(task_id, set()):
        return  # 已识别，跳过
    recognized_students.setdefault(task_id, set()).add(student_id)
    # 写入 attendance_records
    record = AttendanceRecord(
        task_id=task_id,
        student_id=student_id,
        status='present',
        confidence=confidence,
        recognized_at=datetime.now()
    )
    db.session.add(record)
    db.session.commit()
```

### 4.3 考勤结束时补全缺勤记录

```python
def finalize_task(task_id: int):
    task = AttendanceTask.query.get(task_id)
    enrolled = get_enrolled_student_ids(task.course_id)
    recognized = {r.student_id for r in AttendanceRecord.query
                  .filter_by(task_id=task_id).all()}
    absent = enrolled - recognized
    for sid in absent:
        db.session.add(AttendanceRecord(
            task_id=task_id, student_id=sid,
            status='absent', confidence=0.0
        ))
    task.status = 'finished'
    task.end_time = datetime.now()
    task.present_count = len(recognized)
    task.total_students = len(enrolled)
    db.session.commit()
```

### 5.2 特征提取关键实现

```python
# app/ai/face_recognizer.py

class FaceRecognizer:
    def __init__(self, model_type: str, weights_path: str, device: str = 'cpu'):
        self.device = torch.device(device)
        self.model = self._build_model(model_type)
        self.model.load_state_dict(torch.load(weights_path, map_location=self.device))
        self.model.eval()

    def _build_model(self, model_type: str) -> nn.Module:
        if model_type == 'vgg16':
            base = torchvision.models.vgg16(pretrained=False)
            # 去掉 classifier，保留 features + avgpool
            return nn.Sequential(base.features, base.avgpool,
                                 nn.Flatten(), nn.Linear(25088, 512))
        elif model_type == 'resnet50':
            base = torchvision.models.resnet50(pretrained=False)
            base.fc = nn.Linear(2048, 512)
            return base

    @torch.no_grad()
    def extract_features(self, face_tensor: Tensor) -> np.ndarray:
        feat = self.model(face_tensor.unsqueeze(0).to(self.device))
        # L2 归一化
        feat = F.normalize(feat, p=2, dim=1)
        return feat.cpu().numpy().squeeze()
```

### 5.3 余弦相似度匹配算法

```python
# app/ai/feature_database.py

class FeatureDatabase:
    def __init__(self):
        self.student_ids: List[str] = []
        self.feature_matrix: np.ndarray = None  # shape: (N, 512)

    def query(self, query_vec: np.ndarray, threshold_high=0.75,
              threshold_low=0.5) -> dict:
        """
        返回最相似的学生及其状态
        时间复杂度：O(N) 矩阵乘法，N=学生数
        百人规模耗时 < 1ms
        """
        if self.feature_matrix is None or len(self.student_ids) == 0:
            return {"status": "unknown"}

        # 余弦相似度 = 点积（因为已做L2归一化，||a||=||b||=1）
        similarities = np.dot(self.feature_matrix, query_vec)  # shape: (N,)
        best_idx = np.argmax(similarities)
        best_sim = float(similarities[best_idx])

        if best_sim >= threshold_high:
            return {"student_id": self.student_ids[best_idx],
                    "confidence": best_sim, "status": "present"}
        elif best_sim >= threshold_low:
            return {"student_id": self.student_ids[best_idx],
                    "confidence": best_sim, "status": "unverified"}
        else:
            return {"confidence": best_sim, "status": "unknown"}
```

### 5.4 ModelManager 热加载机制

```python
# app/ai/model_manager.py

class ModelManager:
    _instance = None  # 单例

    @classmethod
    def get_instance(cls):
        if cls._instance is None:
            cls._instance = ModelManager(current_app.config['AI_CONFIG'])
        return cls._instance

    def reload_feature_db(self):
        """
        训练完成后调用，无需重启服务
        使用读写锁保护并发安全
        """
        with self._lock:
            new_db = FeatureDatabase()
            new_db.load_from_file(self.config['FEATURE_DB_PATH'])
            self.feature_db = new_db
            current_app.logger.info("Feature database reloaded: %d students",
                                     len(new_db.student_ids))
```

### 6.1 训练配置参数

| 参数 | VGG16 | ResNet50 | 说明 |
|------|-------|----------|------|
| 输入尺寸 | 224×224 | 224×224 | 与预训练一致 |
| Batch Size | 32 | 32 | 视显存调整 |
| 初始学习率 | 1e-4 | 1e-4 | 微调建议小学习率 |
| 冻结层数 | features[0:20] | layer1+layer2 | 保留低层通用特征 |
| Epoch | 30 | 30 | 含 Early Stopping |
| 优化器 | Adam | Adam | |
| 损失函数 | CrossEntropy | CrossEntropy | 分类阶段 |
| 学习率调度 | StepLR(step=10, γ=0.1) | CosineAnnealingLR | |
| 数据增强 | RandomHFlip, ColorJitter, RandomRotation(±15°) | 同左 | 防过拟合 |

### 6.2 训练流程伪代码

```
初始化：加载 ImageNet 预训练权重
阶段1（冻结底层微调，Epoch 1-10）：
  - 冻结 backbone 前N层
  - 只训练新增分类头（FC layers）
  - 学习率 = 1e-3

阶段2（全网端到端微调，Epoch 11-30）：
  - 解冻所有层
  - 学习率降至 1e-4
  - 应用 Early Stopping（patience=5）

每个 Epoch 结束：
  - 在验证集上计算 Top-1 准确率
  - 若为最佳 → 保存 checkpoint

训练完成后：
  - 在测试集上评估（准确率、F1、AUC）
  - 提取所有注册学生特征向量 → 保存 features.npy
  - 调用 ModelManager.reload_feature_db()
```

### 6.3 特征向量构建

```python
def extract_all_features(model, dataloader, student_ids):
    """
    遍历所有学生的人脸图片，对每个学生取多张图片特征的均值向量。
    均值特征比单张更鲁棒（表情、角度变化的平均表示）。
    """
    feature_dict = {}
    for student_id in student_ids:
        imgs = load_student_images(student_id)  # 取所有图片
        feats = [model.extract_features(img) for img in imgs]
        avg_feat = np.mean(feats, axis=0)
        avg_feat = avg_feat / np.linalg.norm(avg_feat)  # 再次归一化
        feature_dict[student_id] = avg_feat
    return feature_dict
```
